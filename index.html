Proyek Aplikasi PPDB Online

Selamat datang di halaman proyek aplikasi PPDB Online.

Aplikasi ini dibangun menggunakan PHP dan memerlukan server web (seperti Apache) serta database (seperti MySQL) untuk dapat berjalan.

Hosting gratis di GitHub Pages ditujukan untuk situs statis (.html, .css, .js) dan tidak dapat menjalankan kode PHP. Oleh karena itu, aplikasi lengkap tidak dapat didemokan di sini.

Untuk melihat kode atau menjalankannya di lingkungan yang sesuai, silakan kunjungi repositori proyek di GitHub.
